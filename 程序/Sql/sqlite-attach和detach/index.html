Sqlite Attach和Detach

sqlite是基于文件的数据库，正常使用sql语句操作一般都是同一个数据库，即同一个文件。若要同时操作多个sqlite数据库呢，这时候就可以用Attach和Detach语法。

Attach 附加数据库，将一个数据库文件附加到当前连接上。

Detach 分离数据库，跟Attach相反，即分离附加的数据库。
